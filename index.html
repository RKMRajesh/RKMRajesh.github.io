Task Management System

A CLI task management tool that lets users create an account to keep track of their tasks. They can then update, view, and delete all tasks current or old. Implemented using C++. Data is stored in JSON files using the nlohmann JSON library.

A simple Shell created for my Operating Systems class. It was Implemented using C and a variety of Linux system calls. It lets the user perform all the basic command line tasks like ls, cd, pwd, cat, etc. I also included a feature for storing commands in single-letter alphabetical variables.
